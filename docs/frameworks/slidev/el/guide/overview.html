Επισκόπηση Διαφανειών

Διαθέσιμο από v0.48.0

Μπορείτε να επισκεφθείτε μια επισκόπηση όλων των διαφανειών σας κάνοντας click στο κουμπίστον πίνακα πλοήγησης και στη συνέχειαστην επάνω δεξιά πλευρά ή πηγαίνοντας απευθείας στο http://localhost:3030/overview.

Η σελίδα επισκόπησης σας παρέχει μια γραμμική λίστα όλων των διαφανειών σας, με όλες τις σημειώσεις σας στο πλάι. Μπορείτε να κάνετε διπλό click στις σημειώσεις για να τις επεξεργαστείτε απευθείας ή να σύρετε τα click sliders για να κάνετε προεπισκόπηση των βημάτων στις διαφάνειές σας. Go to TOC
