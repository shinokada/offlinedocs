Λειτουργία Παρουσιαστή

Κάντε click στοκουμπί στον πίνακα πλοήγησης, ή επισκεφθείτε το http://localhost:3030/presenter χειροκίνητα, για να ενεργοποιήσετε τη λειτουργία παρουσιαστή. Για να παρουσιάσετε, θα πρέπει να ανοίξετε δύο παράθυρα του προγράμματος περιήγησης, ένα για τον παρουσιαστή και ένα για το ακροατήριο. Γενικά, μεγιστοποιώντας το παράθυρο προβολής διαφανειών στην οθόνη του προβολέα σας, και στη συνέχεια ελέγχοντάς το από την οθόνη του φορητού σας υπολογιστή, είναι ο τρόπος με τον οποίο οι άνθρωποι παρουσιάζουν με το Slidev.

Κάθε φορά που αλλάζετε διαφάνειες στη λειτουργία παρουσιαστή, όλα τα άλλα instances σελίδων θα αλλάζουν αυτόματα, ώστε να παραμένουν συγχρονισμένα με τον παρουσιαστή.

Απενεργοποίηση

Η λειτουργία παρουσιαστή είναι ενεργοποιημένη από προεπιλογή.

Μπορείτε να απενεργοποιήσετε αυτή τη λειτουργία με την ακόλουθη ρύθμιση παραμέτρων:

---
presenter: false
---

Διαφορετικά μπορείτε να την ενεργοποιήσετε μόνο για τη λειτουργία dev ή build, ορίζοντας τη λειτουργία που θέλετε στη ρύθμιση παραμέτρων:

---
presenter: dev
---

Σε αυτή την περίπτωση ο παρουσιαστής θα είναι διαθέσιμος μόνο όταν εκτελείται το slidev αλλά όχι όταν εκτελείται το slidev build.

Περιορισμένη απομακρυσμένη πρόσβαση

Μπορείτε να εκτελέσετε την παρουσίασή σας με απομακρυσμένη πρόσβαση εκτελώντας την εντολή slidev --remote.

Αν θέλετε να μοιραστείτε τις διαφάνειές σας με άλλους ανθρώπους, αλλά δεν θέλετε να έχουν πρόσβαση στη λειτουργία παρουσιαστή (είτε επειδή ντρέπεστε για τις σημειώσεις σας είτε επειδή δεν θέλετε να χαλάσουν την παρουσίασή σας), μπορείτε να δώσετε έναν κωδικό πρόσβασης για την προστασία του διακομιστή παρουσιαστή εκτελώντας την εντολή slidev --remote=ο_κωδικός_σας.

Στη συγκεκριμένη περίπτωση, θα πρέπει να δώσετε τον κωδικό πρόσβασης κατά την πρόσβαση στις διαδρομές /presenter/*. Go to TOC
