Ρυθμίστε KaTeX

Δημιουργήστε ./setup/katex.ts με το εξής περιεχόμενο:

import { defineKatexSetup } from '@slidev/types'

export default defineKatexSetup(() => {
  return {
    /* ... */
  }
})

Με το setup, μπορείτε να δώσετε την προσαρμοσμένη ρύθμιση για τις Επιλογές KaTex. Ανατρέξτε στους ορισμούς των τύπων και στις οδηγίες χρήσης τους για περισσότερες λεπτομέρειες. Go to TOC
