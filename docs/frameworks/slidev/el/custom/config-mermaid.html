Ρυθμίστε Mermaid

Δημιουργήστε ./setup/mermaid.ts με το εξής περιεχόμενο:

import { defineMermaidSetup } from '@slidev/types'

export default defineMermaidSetup(() => {
  return {
    theme: 'forest',
  }
})

Με το setup, μπορείτε να δώσετε την προσαρμοσμένη ρύθμιση για το Mermaid. Ανατρέξτε στους ορισμούς των τύπων και στις οδηγίες χρήσης τους για περισσότερες λεπτομέρειες. Go to TOC
