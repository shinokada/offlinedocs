ローカルでサーバーを立ち上げる

npm i -g pnpm

pnpm i
pnpm run dev

コマンドを実行したら http://localhost:3000 にアクセスしてください。

または Vite extension for VS Code をインストールすると、サイドバイサイドで編集できます。
