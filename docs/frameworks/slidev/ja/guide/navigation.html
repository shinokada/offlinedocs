ナビゲーションバー

Slidevのページの左下にマウスを移動すると、ナビゲーションバーが表示されます。

ショートカット ボタン 説明
f フルスクリーンに切り替え
right / space 次のアニメーションもしくはスライドへ
left 前のアニメーションもしくはスライドへ
up - 前のスライドへ
down - 次のスライドへ
o スライドオーバービューに切り替え
d ダークモードに切り替え
- カメラビュー
- レコーディング
- プレゼンターモード
- 統合エディタ
- スライドをダウンロード (SPAビルド
- スライドの情報を表示
- 設定メニューを表示
g - goto...を表示

スライドオーバービュー

oを押すか、ナビゲーションバーの ボタンをクリックすることで、スライドを俯瞰することができ、スライド間を簡単にジャンプできます。
