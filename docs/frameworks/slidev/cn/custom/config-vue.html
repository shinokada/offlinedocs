配置 Vue

Slidev 基于 Vue 3 来渲染应用。你可以针对应用进行扩展，添加自定义插件或自定义配置等操作。

创建 ./setup/main.ts 文件，其内容如下：

import { defineAppSetup } from '@slidev/types'

export default defineAppSetup(({ app, router }) => {
  // Vue App
  app.use(YourPlugin)
})

这也可以作为你 Slidev 应用程序的主入口，在应用启动前做一些初始化操作。
