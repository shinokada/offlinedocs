Monaco 编辑器

每当你想在演示文稿中做一些修改时，只需在 语言id 后添加 {monaco} ——它就会把这个块变成一个功能齐全的 Monaco 编辑器！

```ts {monaco}
console.log('HelloWorld')
```

了解有关 配置 Monaco 的更多信息。

Diff Editor

Monaco 还可以在两个代码快间生成差异。你可以用 {monaco-diff} 将代码块变为一个 Monaco diff 编辑器。你需要用 ~~~ 来分隔原始代码与修改后的代码。

```ts {monaco-diff}
console.log('Original text')
~~~
console.log('Modified text')
```
