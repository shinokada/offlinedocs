Barre de navigation

Déplacez votre souris vers le coin inférieur gauche de la page Slidev, la barre de navigation apparaîtra.

Raccourcis Bouton Description
f basculer en plein écran
droite / espace prochaine animation ou diapositive
gauche animation ou diapositive précédente
haut - diapositive précédente
bas - diapositive suivante
o bascule présentation des diapositives
d basculer le mode sombre
- basculer vue caméra
- enregistrement
- entrer mode présentateur
- toggle éditeurs intégrés
- télécharger les diapositives (n'apparaissent que dans SPA build
- afficher des informations sur les diapositives
- afficher le menu des paramètres
g - montrer goto ...

Présentation des diapositives

En appuyant sur o ou en cliquant sur le bouton dans la barre de navigation, vous pouvez avoir un aperçu de vos diapositives afin de pouvoir passer de l'une à l'autre facilement.
