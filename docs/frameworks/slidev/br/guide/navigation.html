Mova seu mouse para o canto inferior esquerdo da página do Slidev, a barra de navegação aparecerá.

Visualização dos Slides

Ao apertar o ou clicar no botãona barra de navegação, você pode ter uma visão geral dos slides para que você possa alterná-los facilmente.
