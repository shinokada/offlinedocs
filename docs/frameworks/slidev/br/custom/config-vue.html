Configurar Vue

O Slidev usa o Vue 3 para renderizar a aplicação no lado do cliente. Você pode estender a aplicação para adicionar plug-ins e configurações customizadas.

Crie o arquivo ./setup/main.ts com o seguinte conteúdo:

import { defineAppSetup } from '@slidev/types'

export default defineAppSetup(({ app, router }) => {
  // Vue App
  app.use(YourPlugin)
})

Isto pode ser usado também como a entrada principal da sua aplicação do Slidev para fazer algumas inicializações antes da aplicação rodar.
