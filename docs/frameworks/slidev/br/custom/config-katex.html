Configurar KaTeX

Crie o arquivo ./setup/katex.ts com o seguinte conteúdo:

import { defineKatexSetup } from '@slidev/types'

export default defineKatexSetup(() => {
  return {
    /* ... */
  }
})

Com essa configuração, você pode prover as configurações customizadas para o KaTex Options. Consulte as definições de tipo e suas documentações para obter mais detalhes. Go to TOC
