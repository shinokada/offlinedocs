La Propriété buildDir

Définie le répertoire dist pour notre application Nuxt

  • Type : String
  • Par défaut : .nuxt
export default {
  buildDir: 'nuxt-dist'
}

Par défaut, de nombreux outils supposent que .nuxt est un répertoire caché, car son nom commence par un point. Vous pouvez utiliser cette option pour éviter cela. Go to TOC
