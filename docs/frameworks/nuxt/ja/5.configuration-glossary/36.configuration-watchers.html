watchers プロパティ

nuxt.config.js の watchers プロパティは watcher の設定を上書きできます。

  • 型: Object
  • デフォルト: {}

chokidar

  • 型: Object
  • デフォルト: {}

chokidar オプションの詳細は chokidar API を参照してください。

webpack

  • 型: Object
  • デフォルト:
watchers: {
  webpack: {
    aggregateTimeout: 300,
    poll: 1000
  }
}

webpack の watchOptions の詳細は webpack のドキュメントを参照してください。
