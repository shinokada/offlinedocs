ssr プロパティ

デフォルト nuxt ssr の値を変更します。

  • 型: boolean
  • デフォルト: true
  • 設定できる値:
    • true: サーバーサイドレンダリングが有効になります
    • false: サーバーサイドレンダリングを行いません（クライアント側のレンダリングのみとなります）

クライアント側のレンダリングのみ行いたい場合にオプションを false に設定できます

export default {
  ssr: false // サーバーサイドレンダリングを無効化
}
