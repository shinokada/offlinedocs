The modulesDir property

Define the modules directory for your Nuxt application

  • Type: Array
  • Default: ['node_modules']

Used to set the modules directories for path resolving, for example: Webpack's resolveLoading, nodeExternals and postcss. Configuration path is relative to options.rootDir (default: process.cwd()).

export default {
  modulesDir: ['../../node_modules']
}

Setting this field may be necessary if your project is organized as a Yarn workspace-styled mono-repository. Go to TOC
