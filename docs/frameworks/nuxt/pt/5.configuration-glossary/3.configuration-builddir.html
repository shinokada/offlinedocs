A propriedade builDir

Define o diretório de dist (distribuição) para a sua aplicação Nuxt

  • Tipo: String
  • Padrão: .nuxt
export default {
  buildDir: 'nuxt-dist'
}

Por padrão, muitas ferramentas assumem que o diretório .nuxt é um diretório oculto, porque o seu nome começa com um ponto. Você pode usar esta opção para prevenir isso. Go to TOC
