A propriedade watch

A propriedade watch permite você observar ficheiros personalizados para reinicialização do servidor.

  • Tipo: Object
  • Valor padrão: []
watch: ['~/custom/*.js']

O chokidar é usado para definir os observadores. Para saber mais sobre as opções do padrão do chokidar, consulte a API do chokidar. Go to TOC
