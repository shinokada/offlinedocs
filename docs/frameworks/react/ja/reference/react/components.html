"組み込みの React コンポーネント"

React は、JSX で使用できるいくつかの組み込みコンポーネントを公開しています。

組み込みコンポーネント {/built-in-components/}

  • <Fragment> を使い、複数の JSX ノードをまとめることができます。別の書き方として <>...</> があります。
  • <Profiler> を使い、React ツリーのレンダーパフォーマンスをプログラム上で測定することができます。
  • <Suspense> を使い、子コンポーネントがロードされる間、フォールバックを表示することができます。
  • <StrictMode> を使い、バグを早期に見つけるための追加の開発環境専用チェックを有効化します。
