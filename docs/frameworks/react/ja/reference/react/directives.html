"ディレクティブ"

canary: true

これらのディレクティブは、React Server Components を使用している場合や、それらと互換性のあるライブラリを構築している場合にのみ必要です。

ディレクティブによって、React Server Components 互換バンドラに指示を与えます。

ソースコードディレクティブ {/source-code-directives/}

  • 'use client' によりどのコードがクライアント上で実行されるべきかマークします。
  • 'use server' によりクライアント側のコードから呼び出すことができるサーバサイド関数をマークします。 Go to TOC
