### W jaki sposób napisać komentarz w JSX? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* To jest komentarz */}
  Witaj, {name}!
</div>
```

```jsx
<div>
  {/* A to jest 
  wieloliniowy komentarz. */}
  Witaj, {name}! 
</div>
```
