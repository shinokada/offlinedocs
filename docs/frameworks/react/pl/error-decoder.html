Error Decoder

Zdecydowanie zalecamy korzystanie z kompilacji deweloperskiej lokalnie podczas debugowania aplikacji, ponieważ śledzi ona dodatkowe informacje debugowania i zapewnia przydatne ostrzeżenia o potencjalnych problemach w aplikacji.
