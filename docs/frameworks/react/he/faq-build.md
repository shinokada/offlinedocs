### איך אני יכול לכתוב הערות ב-JSX? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* הערות כאן */}
  שלום, {name}!
</div>
```

```jsx
<div>
  {/* זה עובד גם עבור
  .הערות מרובות שורות */}
  שלום, {name}! 
</div>
```
