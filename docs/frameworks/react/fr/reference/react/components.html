"Composants fournis par React"

React met à disposition des composants que vous pouvez utiliser dans votre JSX.

Composants fournis {/built-in-components/}

  • <Fragment>, qu’on peut abréger <>...</>, vous permet de grouper des éléments sans balise enrobante.
  • <Profiler> vous permet de mesurer dans votre code les performances de rendu d'un arbre React.
  • <Suspense> vous permet d'afficher un composant alternatif pendant que les enfants chargent.
  • <StrictMode> active des vérifications supplémentaires spécifiques en mode développement qui vous permettent de repérer des bugs plus tôt.
