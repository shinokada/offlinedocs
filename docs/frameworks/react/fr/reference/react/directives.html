"Directives"

canary: true

Ces directives ne sont utiles que si vous utilisez React Server Components ou créez une bibliothèque compatible avec eux.

Les directives fournissent des instructions à destination des bundlers compatibles avec React Server Components.

Directives dans le code source {/source-code-directives/}

  • 'use client' marque les fichiers sources dont les composants s'exécutent côté client.
  • 'use server' marque les fonctions côté serveur qui peuvent être appelées par du code côté client. Go to TOC
