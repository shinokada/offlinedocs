Configuration

Here's what you need to add to your tauri.conf.json file:

tauri: {
  allowlist: {                  // all API endpoints are default false
    all: false,                 // disable the api
  },
  window: {                     // not yet normative
    glutin: true,
    webview: false
  }
}
