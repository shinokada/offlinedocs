Search

VitePress supports searching your docs site using Algolia DocSearch. Refer their getting started guide. In your .vitepress/config.ts you'll need to provide at least the following to make it work:

import { defineConfig } from 'vitepress'

export default defineConfig({
  themeConfig: {
    algolia: {
      appId: '...',
      apiKey: '...',
      indexName: '...'
    }
  }
})

If you are not eligible for DocSearch, you might wanna use some community plugins like https://github.com/emersonbottero/vitepress-plugin-search or explore some custom solutions on this GitHub thread.

i18n
